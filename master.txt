このコードは、スキーマファイルを読み込んで、マスターのスキーマ、ロードするコードを生成するためのコードです。
このコードで、以下の変更を加えたいので、その変更すべきクラスと変更内容を教えて下さい。

1. guidの型を持ち、autoGeneratedの属性を持つ要素に対して、専用型を定義する
2. その型は public struct [型名] で定義される
3. その型の中でguidの値を保持する
5. その型の中で==など、プリミティブ型で必要な演算子をオーバーロードする
7. 外部キーでそのキーを参照する場合、そのキーの型を持つ変数を持つ
  例えば、現在以下のような生成コードの場合
```cs
namespace Mooresmaster.Model.ItemsModule
{
    public class DataElement
    {
        public readonly global::System.Guid ItemId;
        public readonly string ImagePath;
        public readonly string Name;
        public readonly int MaxStack;

        public DataElement(global::System.Guid ItemId, string ImagePath, string Name, int MaxStack)
        {
            this.ItemId = ItemId;
            this.ImagePath = ImagePath;
            this.Name = Name;
            this.MaxStack = MaxStack;
        }
    }
}

namespace Mooresmaster.Model.BlocksModule
{
    public class FuelSlotElement
    {
        public readonly global::System.Guid ItemId;
        public readonly float Time;
        public readonly float Power;

        public FuelSlotElement(global::System.Guid ItemId, float Time, float Power)
        {
            this.ItemId = ItemId;
            this.Time = Time;
            this.Power = Power;
        }
    }
}
```
